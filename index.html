Desktop Tutorial

Install Cookie Editor on Desktop:

  1. Open the Google Chrome browser on your computer.
  2. Go to the Chrome Web Store.
  3. In the search bar, type "Cookie Editor" and hit Enter.
  4. Find the "Cookie Editor" extension by Christophe Gagnier in the search results and click on it.
  5. Click the "Add to Chrome" button.
  6. A pop-up window will appear â€“ click "Add Extension" to confirm.
  7. Once installed, you will see the Cookie Editor icon (usually a cookie symbol) next to the address bar in Chrome.

Import Cookie using Cookie Editor on Desktop:

  1. In the available platform section, choose the platform you wish to access (e.g., Netflix, Quillbot, Grammarly) and then Copy the JSON.
  2. Navigate to the choosen website where you want to edit cookies, for example Netflix.
  3. Click on the Cookie Editor icon in the Chrome toolbar.
  4. A list of cookies for the current site will appear.
  5. Delete all cookies. To delete all cookie, click the delete button, ensuring that every single one is deleted.
  6. Import Cookie. To Import cookie, click the Import button, paste the copied json and then proceed with the import.
  7. Once the import is completed, a new list of cookies for the current site will be displayed.
  8. Refresh the page to see the effects of your cookie changes.
  9. If the newly imported cookie is valid, you will be automatically redirected to the homepage of the current site.

Mobile Tutorial

Install Cookie Editor in Mobile:

  1. Install a mobile browser that supports extensions, such as Firefox Browser from the Google Play Store.
  2. Open Firefox Browser and Enable Desktop site.
  3. Go to the Chrome Web Store.
  4. Search for "Cookie Editor".
  5. Find the "Cookie Editor" extension by Christophe Gagnier in the search results and click on it.
  6. Click the "Add to Chrome" button.

Import Cookie using Cookie Editor on Mobile:

  1. In the available platform section, choose the platform you wish to access (e.g., Netflix, Quillbot, Grammarly) and then Copy the JSON.
  2. Navigate to the choosen website where you want to edit cookies, for example Netflix.
  3. To use Cookie Editor, tap the three-dot menu in Firefox Browser.
  4. Select "Extensions" from the menu.
  5. Find and tap on "Cookie Editor" in your list of extensions.
  6. The Cookie Editor interface will open, allowing you to view and edit cookies for the current site.
  7. A list of cookies for the current site will appear.
  8. Delete all cookies. To delete all cookie, click the delete button, ensuring that every single one is deleted.
  9. Import Cookie. To Import cookie, click the Import button, paste the copied json and then proceed with the import.
  10. Once the import is completed, a new list of cookies for the current site will be displayed.
  11. Refresh the page to see the effects of your cookie changes.
  12. If the newly imported cookie is valid, you will be automatically redirected to the homepage of the current site.

Dos and Don'ts

Dos

  • Do use Cookie Editor responsibly and ethically.
  • Do respect website terms of service and user agreements.
  • Do keep your browser and extensions up to date for security.
  • Do be cautious when modifying cookies, as it can affect your user experience.
  • Do clear your cookies and cache regularly for privacy and optimal browsing.

Don'ts

  • Don't use Cookie Editor for malicious purposes or to harm others.
  • Don't share or use someone else's personal account information.
  • Don't attempt to bypass paywalls or access unauthorized content.
  • Don't ignore copyright laws and intellectual property rights.
  • Don't use Cookie Editor to impersonate others or for fraudulent activities.
